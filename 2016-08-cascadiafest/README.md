## Networking 101
The 5 layer sandwich:
0. physical layer (hardware)
1. link layer (direct peripherals)
2. network layer (IP)
3. transport layer (UDP, TCP)
4. application layer (HTTP, SMTP, POP3)

---
## Networking 101
Bonus toppings:

6. API layer (Twitter API, GitHub API)
